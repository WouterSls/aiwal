# Dynamic Wallet

Dynamic wallet is wallet toolkit which enables users to quickly create an embedded wallet

## Overview

- Dynamic wallet is wallet provider that creates embedded wallet for user that sign up.
- Users can quickly sign-up through email, socials or EOA
- Dynamic wallet has a Javascript sdk
- Dynamic wallet has "Delegated Access Overview"
